Connection Map Demo

This is a standalone demo of the Connection Map component for Go High Level integration.

How to Embed in Go High Level

  1. Download the connectionMap.js file from this page.
  2. Upload the JS file to your Go High Level site or to a CDN.
  3. Add the following code to your GHL page:
<!-- Connection Map Container -->
<div id="connection-map-container" style="width: 100%; max-width: 800px; margin: 0 auto;"></div>

<!-- Connection Map Script -->
<script src="path/to/connectionMap.js"></script>
<script>
// Initialize the Connection Map when the page is loaded
document.addEventListener('DOMContentLoaded', function() {
ConnectionMap.init();
});
</script>
